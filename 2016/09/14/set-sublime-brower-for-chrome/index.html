为sublime设置快捷键打开浏览器

发表于 2016-09-14   |   分类于 sublime   |  

注意此方法需要SideBarEnhancements插件配合,这里是用的mac下的设置

  • 然后通过ctrl + k, ctrl + b打开侧边栏，在侧边栏的文件中右击，找到 open width -> edit applications
    然后在这里边设置firefox打开的方式。
    application : 路径要修改为自己默认安装的路径。
  • 然后在Key-Bindings-User里设置一下
  • 这里我设置的是alt+s打开chrome浏览器
