Array对象的学习

发表于 2016-01-06   |   分类于 javascript   |  

Array对象的学习

实例方法

push方法

用于在末端添加一个元素

  • 返回值：添加元素后数组的长度
  • 参数：一个或多个元素

也可用于添加元素到一个对象上，但是原对象会变成类数组的对象

  • 新加入的元素的键对应数组的索引
  • 新对象上会生成一个length属性
  1. 参数可以是多个
  2. 返回值是改变后的数组长度
  3. 通过apply方法调用可以用于合并数组
  4. 通赤apply方法调用可以把一个元素添加到对象上，但是对象会变成类数组对象

pop方法

用于删除最后一个元素

  • 返回删除元素后的数组
  • 没有参数

对空数组使用pop方法，不会报错，而是返回undefined。

  1. 没参数
  2. 空数组使用pop不会报错，返回undefined

join方法

  • 返回运算结果组成的字符串
  • 参数为字符串，默认值为逗号

如果数组成呗是undefined 或null 或空位，会被转成空字符串。

通过call方法也可用于字符串。

也可用于类似数组的对象。

  1. 默认值为逗号
  2. undefined和null和空位会被转成空字符串
  3. 可以使用call方法让其用字符串
  4. 也可以用于类数组对象

concat方法

用于多个数组的合并，将新数组成员添加支原数组的尾部

  • 返回一个新数组，原数组不变
  • 参数是一个或多个数组或是其它类型的值
    • 如果参数是数组则直接把新数组成员添到旧数组里
    • 如果参数是数值，则把数值直接添到旧数组里
    • 如果不提供参数，则返回当前数组的一个浅拷贝

concat也可用于将对象合并为数组。

  1. concat 参数有三个种变化
    • 无参
    • 参数是数组
    • 参数是其它类型
  2. 用于将对象合并为数组

shift 方法

用于删除数组的第一个元素，该方法会改变原数组。

  • 返回删除的元素
  • 无参数

和pop方法一样对空数组执行shift方法，不会报错，而会返回undefined

所以可以利用这个特性，用以下方法清空一个数组。

  1. 和pop方法类似的使用方法
  2. 无参数
  3. 空数组调用不会报错，返回undefined

reverse 方法

用于返转数组中元素的顺序

该方法改变原数组

  1. 改变原数组

slice方法

用于提取数组的一部分

  • 返回一个新数组
  • 第一个参数为起始位置，第二个参数为终止位置（不包括该元素），如果省略第二个参数，则一直返回到原数组的最后一个成员
  • 如果参数为负数，则表示倒数计算的位置
  • 如果参数值大于数组成员的个数，或者第二个参数小于第一个参数，则返回空数组。

slice方法的一个重要应用，是把类数组对象转换为真正的数组

  1. 返回一个新数组
  2. 第一个参数为起始位置，第二个位置为终止位置，如果省略第二个参数则一直返回到原数组最后一个成员
  3. 如果参数大于数组成员的个数或第二个参数大于第一个参数，则返回空数组
  4. 可以把类数组对象转换成真正的数组

splice 方法

用于删除数组的一部分成员，并可以在被删除的位置添加新成员

  • 返回值是被删除的元素，该方法会改变原数组

  • 参数

    • 第一个参数–删除的起始位置
    • 第二个参数– 被删除的元素个数
    • 后面的参数– 要插入的新元素

如果只是插入元素，可以把第二个参数设为0

如果只提供第一个参数，等同于在指定位置拆分成两个数组

  1. 用法一：从指定位置删除数组元素个数
  2. 用法二：可以从指定位置拆分数组
  3. 用法三：可以在指定位置插入数组
  4. 用法四：可以同时删除和插入元素

sort 方法

对数组成员进行排序

  • 返回的是排序后的数组
  • 无参数默认是按字典顺序排序

该方法接受一个方法进行自定义排序，可以传入一个函数作为参数，该函数本身接受两个参数，表示进行比较的两个元素。

  • 如果返回值大于0，表示第一个元素排在第二个元素后面;

  • 其化情况下，都是第一个元素排在第二个元素前面。

  1. 默认是按字典顺序排序
  2. 如果函数返回值大于0，第一个元素排在第二个元素后面

map方法

对数组所有成员依次调用一个函数

  • 返回一个新数组。
  • map方法的参数会被传入三个参数，分别element,index,array

map方法不仅可以用数组，还可用于字符串，用来遍历字符串的每个字符。

map方法还可以接受第二个参数，表示回调函数执行时this所指向的对象。

如果数组有空位，map方法的回调函数在这个位置，会跳过数组的空位。但是不会跳过undefined和null

  1. 根据函数结果返回一个新数组
  2. 第一个参数是一个函数接收三个参数ele,index,arr
  3. 第二个参数是要绑定的对象
  4. 会跳过空位，但是不会跳过undefined和null
  5. 可以通过call调用，所以可以用于字符串

forEach 方法

forEach方法和map方法很像，参数一样，对于空位和undefined和null处理方式也是一样，会跳过空位，但不会跳过undefined和null。但是它全map不一样的是没有返回值。

forEach方法无法中断执行，总是会将所有成员遍历完，如果希望符合某种条件时，就中断遍历，要使用for循环。

  1. 所有用法和map方法非常相似，可以参考map
  2. 无返回值
  3. 执行过程不会中断
  4. 如果是类数组对象，则会遍历它的数字键
  5. 也可以遍历字符串

filter 方法

类似map方法

  • 返回结果为true的成员组成的一人新数组
  • 参数同map
  1. 参数同map方法
  2. 返回一个结果为true的新数组

some和every 方法

  • some 是只要有一个数组成员返回值是true，则some方法的返回值就是true
  • every 则是所有成员都是true才返回true
  • 参数和map一样，同样就fn和用来绑定this关键字的对象

reduce和reduceRight 方法

这两个方法依次处理数组的每个成员，最终累计为一个值。只不过两个方法的处理方向不一样。

indexOf和lastIndexOf 方法

返回给定元素在数组中第一次出现的位置，如果没有出现则返回-1

  • 第一个参数是要查询的字符串
  • 第二个参数是搜索开始的位置
  • 这两个方法内部是使用严格相等运算符进行比较的===，而NaN是唯一一个不等于自身的值。

valueOf 方法

该方法返回数组本身

toString 方法

返回数组的字符串形式。

静态方法

Array构造函数

用于生成一个新数组

  • 如果参数是整数正数，表示生成数组的长度，返回的也是新数组的长度
  • 无参数时返回一个空数组
  • 非正整数时，报错
  • 其它参数时，返回由参数组成的新数组
  • 不建议使用构造函数创建数组

isArray 方法

用来判断一个值是否为数组。它可以弥补typeof运算符的不足
