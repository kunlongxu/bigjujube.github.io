String对象的学习

string对象是原生提供的三个包装对象之一

静态方法

fromCharCode方法

  • 返回字符串
  • 一系列的Unicode码点

实例方法

charAt 方法

  • 返回指定位置的字符
  • 参数是从0开始编号的位置
  • 如果参数大于等于字符串的长度，返回空字符串。

该方法完全可以用数组下标代替。

charCodeAt 方法

返回给定位置字符的UniCode码点以十进制表示相当于fromCharCode的逆操作。

  • 如果参数为负数，或大于等于字符串的长度，返回NaN

concat 方法

该方法用于连接两个字符串，返回一个新字符串，不改变原字符串。

  • 该方法在多个参数先将参数转成字符串再连接。
  • 和+不同的是当两个运算数都是数值时，不会转换类型，所以会有不同

slice 方法

用于从原字符串取出子字符串并返回，不改变原字符串。

  • 第一个参数是开始位置
  • 第二个参数是结束位置，结果不包含这个位置
  • 如果参数是负值 或第一个参数大于第二个参数，处理方法同数组的slice方法

substring 方法

作用同slice，但是以下几点不同

  • 如果第一个参数大于第二个参数，会自动更换位置
  • 如果参数是负数，会自动将负数转为0

substr 方法

作用同slice

  • 第一个参数是开始位置
  • 第二个参数是要截取的长度
  • 如果省略第二个参数则截取到原字符串的结束
  • 如果第一个参数是负数，表示倒数
  • 如果第二个参数是负数，将自动转为0，返回空字符串

indexOf，lastIndexOf方法

用于查找一个字符串在另一个字符串中的位置，如果没有找到返回-1，indexOf是从头查找，lastIndexOf从尾部开始匹配。

  • 如果有第二个参数表示匹配的开始位置

trim方法

用于消除字符串两端的空格 制表符 换行符 回车符

无参数，返回的是一个新字符串，不改变原字符串的值。

toLowerCase，toUpperCase方法

用于转换大小写

  • 不改变原字符串
  • 可以通过call调用转换布尔值 或数组

localeCompare 方法

用于比转两个字符串

  • 如果小于0，表示第一个字符串小于第二个字符串
  • 如果等于0，表示两者相等
  • 如果大于0，表示第一个字符串大于第二个字符串

没有什么太大作用

match 方法

用于匹配某个字符串，返回一个数组，成员为第一个字符串，如果没有找到匹配，则返回null

  • 返回的数组还有index属性和input属性
  • index属性表示匹配字符串开始的位置
  • input表示原始字符串

search 方法

用法等同于match但是返回值是匹配的第一个位置，如果没找到返回-1

replace 方法

用于替换匹配的子字符串

第一个参数：旧字符串

第二个参数：新字符串

返回值是改变后的字符串，会改变原字符串

split 方法

按照规则分割字符串，返回一个由分割出来的子字符串组成的数组。

  • 参数是空字符串，则返回成员是原字符串的每一个字符
  • 如果没有参数，则返回数组的唯一成员就是原字符串
  • 如果满足分割的两个字符紧邻着，则返回的数组中会有一个空字符串
  • 如果满足分割的字符串处于开头或结尾，则数组的第一个成员或最后一个成员是空字符串
  • 如果有第二个参数，表示限定返回数组的最大成员数

其中有match,search,replace,split可以用于正则表达式
